# Automation Scripts

A collection of Python scripts for automating common tasks, system monitoring, and fun utilities. These scripts are designed to help with file management, notifications, system checks, and small interactive tools.

## Scripts Included

| Script | Description |
|--------|-------------|
| `file-size-scanner.py` | Scans directories to find files larger than a specified size. Useful for cleaning up storage. |
| `renamefiles.py` | Batch renames files in a folder according to a pattern. |
| `greet.py` | Simple greeting script to welcome users. |
| `simple_chatbot.py` | Basic Python chatbot for text-based interaction. |
| `notification2.py` | Sends notifications on your system for reminders or alerts. |
| `system_status.py` | Monitors system resources and displays CPU, memory, and storage info. |
| `organizer.py` | Organizes files in directories into categorized folders automatically. |
| `weather.py` | Fetches current weather information using online APIs. |
